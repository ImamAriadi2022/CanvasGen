# Berkas Submisi Dicoding - CanvasGen

Folder `submit/` ini berisi seluruh dokumen dan berkas submission yang diperlukan untuk penilaian Dicoding pada program beasiswa IDCamp / BFG AI.

---

## Isi Folder Submisi (`submit/`)

1. **`Pipeline_submission_BFGAI_Imam.ipynb`**: Notebook laporan eksekusi pipeline machine learning Stable Diffusion 1.5, komparasi scheduler, inpainting, dan outpainting.
2. **`Streamlit_submission_BFGAI_Imam.ipynb`**: Notebook alur instalasi, pembuatan aplikasi web Streamlit, dan workflow deployment.
3. **`requirements.txt`**: Daftar dependensi terikat versi.
4. **`README.md`**: Panduan pengujian dan evaluasi submission ini.
5. **`submission_checklist.md`**: Tabel periksa kelengkapan prasyarat Dicoding.

---

## Instruksi Pengujian untuk Reviewer Dicoding

1. Ekstrak file zip submission ini.
2. Pasang dependensi menggunakan `pip install -r requirements.txt`.
3. Jalankan pengujian unit menggunakan `python -m unittest discover tests` atau `pytest`.
4. Jalankan aplikasi Streamlit dengan perintah `streamlit run app.py`.
